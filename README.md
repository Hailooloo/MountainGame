# 山中一甲子 - 五行放置游戏

一款基于中国传统五行哲学和易学智慧的休闲放置类游戏。

## 游戏概述

**世界观**：玩家拥有一座基于"地山谦"卦象的荒山，目标是通过调节"金木水火土"的平衡，让荒山长出灵植、吸引神兽，最终变成世外桃源。

## 核心机制

### 五行流转系统
- **木（生长/技能）**：决定植物的生长速度
- **火（能量/名声）**：决定荒山的温度与吸引神兽的概率
- **土（承载/防御）**：决定土地肥力与存储空间
- **金（技术/工具）**：决定玩家采集资源的效率与工具升级
- **水（财富/流通）**：作为核心货币，调节火的燥气

### 五行相生相克
- **相生**：木生火 → 火生土 → 土生金 → 金生水 → 水生木
- **相克**：木克土 → 土克水 → 水克火 → 火克金 → 金克木

## AI互动功能

输入你的每日情绪（如"今天工作很累"），由山灵（AI）生成一段带有易学智慧的治愈系台词，并触发对应的天气变化：
- 疲劳 → 壬水雨（增加水元素）
- 快乐 → 丙火日（增加火元素）
- 压力 → 甲木风（增加木元素）
- 平静 → 戊土雾（增加土元素）
- 创意 → 庚金晴（增加金元素）

## 游戏特色

1. **五行平衡系统**：实时计算五行平衡状态，影响游戏进程
2. **动态荒山**：灵植和神兽随五行变化而生长来访
3. **时间流逝**：游戏内时间流转，季节更替
4. **天气系统**：AI对话触发特殊天气效果
5. **本地保存**：游戏进度自动保存到浏览器
6. **响应式设计**：适配各种屏幕尺寸

## 如何开始

### 本地运行
1. 直接用浏览器打开 `index.html` 文件
2. 游戏会自动加载，无需额外安装

### 在线游玩（推荐）
游戏已部署到 GitHub Pages，可通过以下方式访问：
1. **GitHub Pages 地址**：`https://[你的用户名].github.io/[仓库名]`
2. **自定义域名**：可设置 CNAME 指向游戏页面

## 部署到 GitHub Pages

### 步骤一：创建 GitHub 仓库
1. 访问 [GitHub](https://github.com) 并登录
2. 点击右上角 "+" → "New repository"
3. 输入仓库名称（如 `mountain-game`）
4. **重要**：选择 **Public**（公开）
5. 点击 "Create repository"

### 步骤二：上传游戏文件
```bash
# 使用 Git 命令行
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/[你的用户名]/[仓库名].git
git push -u origin main

# 或者使用 GitHub Desktop 客户端
```

### 步骤三：启用 GitHub Pages
1. 进入仓库页面
2. 点击 "Settings" → "Pages"
3. 在 "Source" 部分选择：
   - Branch: `main`
   - Folder: `/ (root)`
4. 点击 "Save"

### 步骤四：等待部署完成
1. GitHub 会自动部署（约1-2分钟）
2. 刷新页面，你会看到 "Your site is published at..." 的链接
3. 点击链接即可在线游玩

### 步骤五：自定义域名（可选）
1. 在 "Settings" → "Pages" 的 "Custom domain" 处输入你的域名
2. 按照提示配置 DNS 记录

## 游戏操作

### 基本操作
- 点击五行卡片下方的按钮调节对应元素
- 在"山中悟道"区域输入心情与山灵对话
- 使用右下角按钮保存/加载/重置游戏

### 快捷键
- `Ctrl + S`：保存游戏
- `Ctrl + L`：加载游戏
- `Ctrl + R`：重置游戏
- `Esc`：关闭帮助窗口

## 游戏目标

1. **短期目标**：保持五行平衡，获得每日奖励
2. **中期目标**：让荒山长出20株灵植，吸引10只神兽
3. **长期目标**：将荒山建设成世外桃源（五行全部达到90+）
