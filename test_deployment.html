山中一甲子 - 部署测试

1 检查文件完整性

验证游戏所需的所有文件是否就绪：

✅ index.html - 主游戏文件
✅ styles.css - 样式文件
✅ game.js - 游戏逻辑
✅ ai-service.js - AI服务
✅ visual-effects.js - 视觉特效
✅ victory-screen.js - 胜利画面
✅ .gitignore - Git忽略文件
✅ .github/workflows/deploy.yml - 部署配置
2 GitHub 仓库设置

确保仓库设置为：

  • ✅ 公开仓库 (Public)
  • ✅ 主分支为 main
  • ✅ 启用 GitHub Pages
3 GitHub Pages 配置

访问仓库 Settings → Pages：

Source: Deploy from a branch
Branch: main
Folder: / (root)
4 访问地址

游戏发布后可通过以下地址访问：

https://[你的用户名].github.io/[仓库名]/

⚠️ 首次部署需要等待1-2分钟

5 多设备访问

成功部署后，可在以下设备访问：

  • 📱 手机：打开浏览器，输入游戏地址
  • 💻 电脑：任何电脑浏览器均可访问
  • 📱 平板：适配各种屏幕尺寸

🎮 所有设备数据独立保存（浏览器本地存储）
